# Study Log Sync - 学習記録・目標達成アプリ

「Study Log Sync」は、社会人学生向けに開発された**学習計画・集中タイマー・記録管理を一元化したWebアプリケーション**です。
1つのHTMLファイルだけで動作し、データは全てブラウザ内に保存されるため、会員登録なしですぐに使えます。

## 🌟 アプリの概要

- **目的**: 忙しい日々のスキマ時間を可視化し、学習の継続をサポートする。
- **特徴**:
  - **Plan (計画)**: やるべき学習タスクを登録・管理。
  - **Focus (集中)**: ポモドーロタイマーで目の前の課題に集中。
  - **Log (記録)**: 学習の積み上げをグラフと履歴で振り返り。

---

## 🚀 使い方

アプリは3つのタブ（画面）で構成されています。画面下部のメニューで切り替えます。

### 1. 📝 Plan (計画画面)

学習予定を立てる画面です。

1. **タスクの追加**:
   - リストから科目・カテゴリを選択します（「＋」ボタンで新しい科目も追加可能）。
   - 必要に応じて「サブカテゴリ」（例：第3章、英単語など）を入力します。
   - 学習内容を入力し、「リストに追加」ボタンを押します。
2. **学習の開始**:
   - リストに追加されたタスクの「▶ 学習を開始する」ボタンを押すと、そのタスク情報を持って**Focus画面**へ移動します。
3. **完了管理**:
   - 完了したタスクは、下部の「完了済みタスク」エリアに移動され、達成感を感じられます。

### 2. ⏱️ Focus (集中画面)

ポモドーロテクニック（25分集中＋5分休憩）を活用して学習する画面です。

1. **タイマー操作**:
   - `Start` でカウントダウン開始、 `Pause` で一時停止、 `Reset` で最初に戻ります。
   - 下の「25分」「5分」ボタンで時間を切り替えられます。
2. **記録の保存**:
   - タイマー終了後、または任意のタイミングで、実際に学習した時間を確認します。
   - 「今日の一言メモ」を入力し、「記録を保存して完了にする」ボタンを押すと、**Log画面**にデータが保存されます。

### 3. 📊 Log (記録・振り返り画面)

頑張った成果を確認する画面です。

1. **進捗確認**:
   - 「今週の目標（10時間）」に対する進捗バーが表示されます。
   - 直近の合計時間が60分を超えると、応援メッセージが表示されます。
2. **履歴管理**:
   - 日付、科目、サブカテゴリ、学習時間、メモがリスト形式で表示されます。
   - 間違えた記録は「削除」ボタンで消せます。
3. **データ管理（バックアップ）**:
   - 画面最下部の「設定・データ管理」から、学習データをJSONファイルとしてPCに保存したり、復元したりできます。
   - 機種変更やブラウザのキャッシュクリア対策として定期的な保存をお勧めします。

## 🛠️ 実装の仕組み (Technical Details)

本アプリは、以下の技術を用いて実装されています。

### 1. シングルページアプリケーション (SPA)

- **構造**: `index.html` 1ファイルのみで動作します。
- **仕組み**: JavaScriptによって、「Plan」「Focus」「Log」の3つのセクション（`<section>`タグ）の表示/非表示を瞬時に切り替えることで、アプリのような画面遷移を実現しています。

### 2. データ保存 (LocalStorage)

- サーバー（データベース）を使わず、ブラウザの**LocalStorage**という機能を利用しています。
- **メリット**: ネット環境がなくても動作し、データを外部に送信しないためプライバシーも安全です。
- **デメリット**: ブラウザの履歴を削除するとデータも消えてしまうため、「JSONエクスポート機能」を実装して対策しています。

### 3. データ構造 (JSON)

内部では以下のようなデータ構造で管理しています。

- `tasks`: 未完了・完了タスクの配列
- `logs`: 学習完了ログの配列
- `categories`: ユーザー定義の科目リスト

### 4. 開発言語

- **HTML5**: セマンティックなマークアップ
- **CSS3**: 変数（CSS Variables）を使用したテーマカラー管理、モバイルファーストなレスポンシブデザイン
- **JavaScript (ES6+)**: DOM操作、イベントリスナー、LocalStorage操作

## 📦 GitHub Pages への公開手順 (大学課題用)

1. GitHubリポジトリを作成します。
2. このフォルダ内の `index.html` と `README.md` をアップロードします。
3. リポジトリの **Settings > Pages** に移動します。
4. **Source** で `main` ブランチを選択し、Saveします。
5. 数分後、発行されたURLにアクセスするとアプリが利用できます。
